Shortcut Keys

The following shortcut keys can be used to access ButtercupReader functionality.

O Open Dialog
C Show Content
I Show Book Info
Shift+? Show Help
Spacebar Play or Pause
Home Go to First Heading
End Go to Last Heading
Page Up Go to Previous Page
Page Down Go to Next Page
Up Arrow Go to Previous Header
Down Arrow Go to Next Header
Left Arrow Go to Previous Phrase
Right Arrow Go to Next Phrase
K Increase Volume
J Decrease Volume
M Mute or Un-Mute
F Find Text
P Go To Page
Shift+K Increase Content Size
Shift+J Decrease Content Size
A Contrast
Escape Close Panel
Tab Tab Forward
Shift+Tab Tab Backwards
Enter Start Action
Backspace Clear Search Textbox
Ctrl+Space Toggle Self Voicing
Numpad * Speak Help Text
Numpad - Stop Self Voice

Enabling Text-to-Speech and Self Voicing (Internet Explorer only)

ButtercupReader uses the Microsoft Speech API (SAPI) to enable self-voicing through text-to-speech. For this to work within your browser you will need to make a small change to your security settings.

Note, self-voicing currently only works in Internet Explorer running on Windows.

  1. From your browser’s "Tools" menu select "Internet Options"
  2. Select the "Security" tab
  3. Now you can add ButtercupReader to your browser’s trusted sites as follows:
    1. Select the "Trusted Sites" option in the "Select a zone to view or change security settings" section, then click the "Sites" button
    2. A details panel appears. Click "Add" to add ButtercupReader to the list of Trusted Sites, then click "Close"
  4. Now you can check your ActiveX settings as follows:
    1. Click the "Custom level" button within the "Security level for this zone" section
    2. Scroll or tab down to the "ActiveX controls and plug-ins" section
    3. Ensure the "Initailize and script ActiveX controls not marked as safe for scripting" option is enabled, then click "OK"
  5. Then click "OK" to exit the Internet Options panel
  6. Finally, refresh the web page
