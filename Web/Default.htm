The Online Digital Talking Book Reader (Demo)

What is ButtercupReader?

It's an early demo of a web based application which enables you to read, listen to & navigate through 'DAISY 3' digital talking books online.

  • Read the User Guide
  • Open ButtercupReader
  • Read 'Peter Rabbit' (Sample)
  • Read 'UIA Wikipedia'

If you're using Internet Explorer (on Windows), and want Buttercup to self-voice (or you're playing a book without embedded audio), then you'll need to change your web browser settings. See User Guide for more details.
